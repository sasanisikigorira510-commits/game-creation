# 最初にやること

この Unity プロジェクトは土台だけ配置済みです。

## すぐやること

1. Unity で `WitchTowerGame` を開く
2. コンパイルエラーが出たら内容を確認する
3. `TextMeshPro Essential Resources` を import する
4. `Assets/Scenes` に以下の4シーンを作る
   - `BootScene`
   - `TitleScene`
   - `HomeScene`
   - `BattleScene`
5. `Assets/Resources/MasterData` に `MasterDataRoot.asset` を作る
6. `Assets/MasterData` 配下に各 `ScriptableObject` を作る
