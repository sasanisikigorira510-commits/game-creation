# MVP進捗メモ v0.1

## 実装済みの土台
- 起動、タイトル、ホーム、バトルのシーン構成
- Resources経由のマスターデータ自動読込
- セーブ/ロード
- 自動戦闘
- スキル3種
- 強化
- 装備変更
- 勝利報酬
- レベルアップ
- リザルト表示
- 敵特性
- 簡易ダメージ演出
- デイリー報酬
- ミッション
- 放置報酬

## まだ Unity で配線が必要なもの
- 各シーンの GameObject 配置
- UI の Text / Button / Panel 紐づけ
- ScriptableObject 実データ作成
- MasterDataManager 参照設定

## まだ簡易実装のもの
- スキル効果はコード固定
- 敵AIは単純
- ミッションは2種だけ
- リザルト画面は最小表示
- 装備一覧は簡易切り替え前提
