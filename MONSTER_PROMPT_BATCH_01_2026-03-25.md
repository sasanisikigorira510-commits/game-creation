# Monster Prompt Batch 01

## 使い方
- まずは `1体ずつ` ChatGPT に依頼する
- 形式は `1枚のモンスターシート` に統一する
- 文字やファイル名は画像内に入れない
- 背景は `白無地` でよい
- 上段に `小さいバトル用ポーズ4つ`
- 下段中央に `大きいポートレート1つ`
- 今の切り出し処理は [ProcessMonsterSheet.ps1](/C:/Users/sasan/OneDrive/デスクトップ/ゲーム作成/tools/ProcessMonsterSheet.ps1) を使う

## 共通依頼文
```text
ダークファンタジー風のスマホゲーム用モンスター画像を1枚で作成してください。

条件
- 画像は1枚のみ
- 背景は白無地
- 画像内に文字、ファイル名、説明文は入れない
- 上段に小さいバトル用ポーズを4つ横並びで配置
- 下段中央に大きいUIポートレートを1つ配置
- 上段4ポーズは32x32ドット絵に切り出しやすいよう、余白を十分に空ける
- 下段ポートレートは全身または自然な上半身が切れずに入るようにする
- スマホゲーム向けで見やすいシルエットにする
- ダークファンタジーの世界観に合わせる
- 細かすぎる背景装飾は入れない
```
