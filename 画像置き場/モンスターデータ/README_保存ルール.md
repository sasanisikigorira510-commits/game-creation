# モンスターデータ保存ルール

## まずここに保存
- 新しく ChatGPT で作ってもらった画像は `00_INBOX` に入れる
- 1体ごとに用意してあるサブフォルダへ保存する

## フォルダの意味
- `00_INBOX`
  - 新しく保存した画像をまず置く場所
- `10_SOURCE_SHEETS`
  - 採用した元シート
  - 切り出し前の原本
- `20_PROCESSED`
  - こちらで切り出した完成PNG
- `99_ARCHIVE`
  - HTML保存や会話保存などの資料

## おすすめ運用
1. ChatGPT で画像ができたら、対応する `00_INBOX` のフォルダに保存
2. 画像が複数できても、そのまま全部入れてよい
3. 保存したら、どのモンスターの画像を入れたかだけこちらに伝える
4. こちらで確認して `10_SOURCE_SHEETS` と `20_PROCESSED` に整理する
