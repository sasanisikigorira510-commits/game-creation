# 使い方

## 重要
- `新しいチャット` で使う
- `1体につき1ファイル` だけ使う
- `ロックゴーレムの画像` や過去の会話は添付しない
- バッチ資料を丸ごと貼らず、対象モンスターのファイルだけ送る

## 送るときの一言
```text
今回はこのモンスターだけを作成してください。
以前に作ったロックゴーレムは参照せず、下の条件だけで新規に生成してください。
```

## 保存先
- できた画像は対応する `00_INBOX` のフォルダへ保存
- 例:
  - ゴブリン: [00_INBOX/goblin](/C:/Users/sasan/OneDrive/デスクトップ/ゲーム作成/画像置き場/モンスターデータ/00_INBOX/goblin)
